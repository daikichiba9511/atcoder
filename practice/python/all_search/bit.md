## bit DP

- n個の要素の順列としてありうるものの中から最適な物を求めたい時に使える。

- dp[S] $\coloneqq$ 全体集合の中の真の部分集合{1, 2, ... , n-1}についてその中で順序を最適化した時のSの中での最小コスト。
